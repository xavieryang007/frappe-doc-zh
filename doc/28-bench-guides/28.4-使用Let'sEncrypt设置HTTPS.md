# 28.4 使用Let's Encrypt设置HTTPS

## 概述
本指南详细介绍了如何在Frappe框架环境中，通过Let's Encrypt为您的站点自动配置HTTPS（SSL/TLS加密）。

## 核心步骤

### 1. 前提条件
- 确保您的域名已正确解析到服务器IP
- 确保服务器上的80和443端口对外开放

### 2. 使用Bench命令自动化配置
Frappe的`bench` CLI工具提供了简化的命令来集成Let's Encrypt

#### 基本命令格式：
```bash
bench setup ssl [你的域名]
```

或者

```bash
bench setup lets-encrypt [站点名称]
```

此命令会自动处理证书申请、验证和Web服务器（通常是Nginx）配置

### 3. 验证与生效
- 命令执行成功后，它会自动更新站点的Nginx配置文件，重定向HTTP流量到HTTPS
- 您应该访问您的站点，确认浏览器地址栏显示安全锁标志

### 4. 证书自动续期
- Let's Encrypt证书有效期为90天
- `bench`通常已设置自动续期任务（例如通过cron作业），无需手动干预

### 5. 高级配置（可选）
- 指南可能还包含如何处理多个域名（SAN证书）或自定义配置的说明

## 重要说明

### 生产环境
- 此方法专为生产环境设计，是确保站点通信安全的推荐做法

### 依赖关系
- 该功能依赖于`certbot`（Let's Encrypt的官方客户端）和正确的Nginx配置
- `bench`脚本会尝试自动处理这些依赖
