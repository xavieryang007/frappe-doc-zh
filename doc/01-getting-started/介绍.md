# 介绍 (Introduction)

## 概述

Frappe框架是一个全栈、功能完备的Web框架，基于Python和JavaScript开发，使用MariaDB作为数据库。它是支撑ERPNext系统的底层框架，但本身是通用的，可用于构建任何数据库驱动的应用程序。

## 核心特点

- **元数据即数据**：Frappe将元数据（如表单结构、权限设置）也视为数据，使得前端开发更为简便。
- **全栈架构**：提供前后端一体化的解决方案，内置管理界面（Desk），支持表单、列表、权限、文件管理等常见功能。
- **开箱即用**：无需额外配置即可快速搭建具备用户管理、REST API、自定义报表等功能的应用。

## 为什么选择Frappe？

- 适合需要快速构建企业级应用（如ERP、CRM）的开发者。
- 内置丰富的UI组件和权限模型，减少重复开发工作。
