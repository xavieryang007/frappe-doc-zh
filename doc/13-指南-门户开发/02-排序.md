# 13.2 排序 (Ordering)

## 页面排序 (Ordering)

在门户开发中，您可以通过定义 `index.txt` 文件来指定页面在索引中的显示顺序。

### 操作方法
1. 在存放页面的文件夹中创建名为 `index.txt` 的文件。
2. 在该文件中按顺序列出各个页面的名称（**不包含扩展名**），每行一个。

### 示例
例如，对于当前指南所在的文件夹，`index.txt` 文件内容如下：

```
adding-pages
ordering
contents
context
building
```

这样，页面在门户索引中将按照 `index.txt` 中定义的顺序显示。
