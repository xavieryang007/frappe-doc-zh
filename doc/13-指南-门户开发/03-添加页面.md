# 13.3 添加页面 (Adding Pages)

## 添加页面

要添加页面，只需在 `www` 文件夹中添加 `.html` 或 `.md` 文件。这些页面只能包含内容，不能包含 `<html>` 和 `<body>` 标签。

您也可以编写 Markdown 页面。

### 索引

文件夹中的第一个文件必须命名为 `index.md` 或 `index.html`。
必须存在这两个文件之一，系统才能将此文件夹视为构建页面的有效文件夹。

### Markdown 示例

```
# 这是一个标题

这是一些页面内容 [一个链接](/home/link)
```

### 添加链接

指向页面的链接 URL 可以省略 `.html` 扩展名，例如：`/home/link`。

### 标题

文件中第一个出现的 `<h1>` 区块（如果存在）将作为页面标题，除非在特殊标签中另行指定。如果未指定 `<h1>` 或标题，则文件名将作为标题。

### 添加 CSS

您还可以添加一个同名的 `.css` 文件（例如，`index.md` 对应 `index.css`），该 CSS 文件将与页面一同渲染。

### 特殊标签

1.  `<html>` 标签将使页面使用 Jinja 模板渲染。
2.  `<title>` 标签将添加自定义标题。
3.  `<nocache>` 标签将不在页面中添加面包屑导航。
4.  `<cache>` 标签将启用缓存（如果您使用了 Jinja 模板）。

这些是"门户开发指南"中关于"添加页面"的核心内容要点。
