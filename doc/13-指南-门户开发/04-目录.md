# 13.4 目录 (Table of Contents)

## 目录功能（Table of Contents）

### 1. 功能说明
- 在门户页面中，可通过插入 `{index}` 字符串自动生成带编号的页面目录列表。
- 该功能仅适用于 `index.md` 或 `index.html` 文件。

### 2. 使用方法
- 在内容中需要显示目录的位置，单独起一行并写入 `{index}`。
- 示例：
  ```
  # 页面标题
  
  这里是正文内容。
  
  ### 目录：
  {index}
  ```
  渲染后会显示为当前空间所有页面的编号列表。

### 3. 前后导航按钮
- 使用 `{next}` 可生成"下一页"按钮。
- 使用 `{previous}` 可生成"上一页"按钮。
- 示例：
  ```
  {previous}
  {next}
  ```
  这些按钮会自动链接到当前页面的前一篇或后一篇文档。
