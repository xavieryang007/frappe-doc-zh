# 15.3 OpenID Connect和Frappe社交登录

## 概述
OpenID Connect (OIDC)是基于OAuth 2.0的身份验证协议，用于实现标准化的社交登录功能。

## 支持的身份提供商
- Google
- Facebook
- GitHub
- Microsoft
- 其他OIDC兼容提供商

## 配置步骤
### 1. 身份提供商配置
- 在提供商平台注册应用
- 获取客户端ID和密钥
- 配置回调URL

### 2. Frappe配置
- 安装社交登录应用
- 配置提供商信息
- 设置用户映射规则

## 代码示例
```python
# OpenID Connect集成
from frappe.integrations.oauth2 import OAuth2

class OpenIDConnect(OAuth2):
    def __init__(self, provider):
        super().__init__(provider)
    
    def get_user_info(self, access_token):
        # 获取用户信息
        return self.get("userinfo", token=access_token)
```

## 用户流程
1. 用户点击社交登录按钮
2. 重定向到身份提供商
3. 用户授权应用
4. 回调到Frappe应用
5. 创建或登录用户账户
