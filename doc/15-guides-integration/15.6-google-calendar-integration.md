# 15.6 Google日历集成

## 概述
Google日历集成允许Frappe应用与Google Calendar进行双向同步，实现日程管理的无缝对接。

## 核心功能
- 日历事件同步
- 会议室预订管理
- 资源调度优化
- 实时通知推送

## 集成步骤
### 1. Google API配置
- 启用Google Calendar API
- 创建服务账户或OAuth 2.0凭据
- 配置API访问权限

### 2. Frappe配置
- 安装日历集成应用
- 配置认证信息
- 设置同步规则和频率

## 代码示例
```python
# Google日历集成示例
from frappe.integrations.google_calendar import GoogleCalendar

# 初始化日历客户端
calendar = GoogleCalendar()

# 创建事件
def create_calendar_event(event_data):
    return calendar.insert_event(event_data)

# 获取事件列表
def get_events(time_min, time_max):
    return calendar.list_events(time_min, time_max)

# 同步日历
def sync_calendar():
    # 从Google获取事件
    google_events = calendar.get_events()
    # 同步到Frappe
    sync_to_frappe(google_events)
```

## 同步策略
- 双向同步（可选）
- 冲突解决规则
- 增量同步优化
- 错误重试机制
