# 15.4 Webhooks

## 概述
Webhooks允许应用在特定事件发生时向外部系统发送实时通知，实现事件驱动的集成。

## Webhook类型
- 出站Webhook (Outbound Webhooks)
- 入站Webhook (Inbound Webhooks)
- 双向Webhook (Bidirectional Webhooks)

## 配置流程
### 1. 创建Webhook
- 定义触发事件
- 配置目标URL
- 设置请求格式
- 配置重试策略

### 2. 安全配置
- 添加签名验证
- 配置IP白名单
- 设置请求速率限制

## 代码示例
```python
# Webhook处理示例
import frappe
from frappe.webhooks import trigger_webhook

# 触发Webhook
def trigger_customer_created(customer):
    trigger_webhook("customer.created", customer)

# 处理入站Webhook
@frappe.whitelist(allow_guest=True)
def handle_webhook(data):
    # 验证签名
    if not verify_signature(data):
        return "Invalid signature"
    
    # 处理数据
    process_webhook_data(data)
    return "Success"
```

## 使用场景
- 订单状态更新通知
- 库存变化同步
- 支付状态回调
- 用户注册事件
