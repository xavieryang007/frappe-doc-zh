# 15.5 社交登录密钥

## 概述
社交登录密钥用于配置和管理第三方身份提供商的认证信息，实现一键式社交登录功能。

## 支持的提供商
- Google
- Facebook
- GitHub
- LinkedIn
- Twitter
- 自定义OAuth 2.0提供商

## 密钥配置
### 1. 提供商设置
- 在提供商平台创建应用
- 获取客户端ID和密钥
- 配置授权范围和权限

### 2. Frappe配置
- 创建社交登录密钥文档
- 填写提供商信息
- 设置回调URL
- 配置用户映射规则

## 代码示例
```python
# 社交登录密钥管理
import frappe
from frappe.integrations.doctype.social_login_key.social_login_key import get_social_login_key

# 获取社交登录密钥
def get_google_login_key():
    return get_social_login_key("Google")

# 验证社交登录
def validate_social_login(provider, user_data):
    key = get_social_login_key(provider)
    if not key.enabled:
        return False
    
    # 验证用户数据
    return validate_user_data(user_data, key)
```

## 安全注意事项
- 妥善保管客户端密钥
- 定期轮换密钥
- 限制授权范围到最小必要权限
- 监控登录活动
