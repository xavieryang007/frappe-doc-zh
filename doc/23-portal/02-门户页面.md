# 门户页面 (Portal Pages)

门户页面是Frappe框架中用于创建公共可访问页面的功能模块，允许用户创建自定义的Web页面。

## 功能特性

- **可视化编辑器**: 支持拖拽式页面构建
- **响应式设计**: 自动适配不同设备屏幕
- **模板系统**: 提供丰富的页面模板
- **SEO优化**: 支持搜索引擎优化配置
- **多语言支持**: 内置国际化支持

## 创建门户页面

### 基本页面创建

```python
# 创建门户页面
portal_page = frappe.get_doc({
    "doctype": "Portal Page",
    "title": "关于我们",
    "route": "about-us",
    "published": 1,
    "content": "<h1>关于我们</h1><p>公司介绍内容...</p>"
})
portal_page.insert()
```

### 使用页面模板

```python
# 使用预定义模板创建页面
portal_page = frappe.get_doc({
    "doctype": "Portal Page",
    "title": "联系我们",
    "route": "contact",
    "template": "contact_form",
    "published": 1
})
portal_page.insert()
```

## 页面管理

### 页面属性

- **标题**: 页面的显示标题
- **路由**: 页面的URL路径
- **模板**: 使用的页面模板
- **发布状态**: 控制页面是否对外可见
- **元数据**: SEO相关的元信息

### 页面内容编辑

支持多种内容编辑方式：

1. **HTML编辑器**: 直接编辑HTML代码
2. **Markdown编辑器**: 使用Markdown语法
3. **可视化编辑器**: 所见即所得的编辑体验

门户页面支持动态内容渲染：

```jinja2
<!-- 使用Jinja模板语法 -->

```

## 权限控制

### 页面访问权限

- **公开页面**: 任何人都可以访问
- **受保护页面**: 需要登录才能访问
- **角色限制**: 基于用户角色的访问控制

### 编辑权限

- **管理员**: 可以创建、编辑、删除任何页面
- **编辑者**: 只能编辑自己创建的页面
- **查看者**: 只能查看页面内容

## 最佳实践

1. **URL规划**: 设计清晰、有意义的URL路由
2. **性能优化**: 对大型页面进行分块加载
3. **缓存策略**: 合理使用缓存提高页面加载速度
4. **移动端适配**: 确保页面在移动设备上的良好体验
