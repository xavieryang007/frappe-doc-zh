# 21.1 工作区

工作区是用户登录后首先看到的页面，并拥有一个专用的侧边栏，用户可以根据需要调整工作区的位置。工作区本身是一个用于构建用户特定页面的工具，包含一些标准工作区（位于侧边栏的"公共"部分，对所有用户可见），用户也可以创建私有工作区（仅对当前登录用户可见，位于"我的工作区"部分）。

## 功能说明

### 1. 工作区类型
- **公共工作区**：所有用户可见，通常由具有"工作区管理员"角色的用户创建和管理。
- **私有工作区**：仅创建者（当前登录用户）可见，位于"我的工作区"部分。

### 2. 工作区块（Workspace Blocks）
- 工作区块是构建工作区页面的基本单元，可以通过不同排列方式组合，满足特定用例的需求。
- 支持多种块类型，如图表、快捷方式、笔记等。

### 3. 侧边栏（Sidebar）
- 侧边栏分为"我的工作区"和"公共"两部分。
- 用户可以在侧边栏中拖拽调整工作区顺序。

### 4. 自定义功能
- 工作区有两种模式：**只读模式**和**编辑模式**。
- 用户可以通过点击"编辑"按钮自定义工作区内容和布局。
- 推荐使用工作区构建器UI进行所有自定义操作，避免直接修改文档块导致布局问题。

## 使用方法

### 1. 创建新工作区
- 点击"创建工作区"按钮。
- 输入标题，点击"创建"。

### 2. 创建子工作区
- 点击"创建工作区"按钮。
- 输入标题，在"父级"字段中选择"Home"或其他工作区作为父级，点击"创建"。

### 3. 自定义工作区
- 进入工作区页面，点击"编辑"按钮进入编辑模式。
- 添加、删除或调整工作区块的位置和内容。
- 保存更改后，工作区将根据新布局显示。

### 4. 管理侧边栏
- 在侧边栏中拖拽工作区项调整顺序。
- 仅"工作区管理员"可以创建、编辑或删除公共工作区。

## 注意事项
- 直接在工作区文档中添加新块可能导致布局错乱，建议始终使用工作区构建器UI进行操作。
- 私有工作区仅对创建者可见，适合个人任务管理；公共工作区适用于团队协作。
