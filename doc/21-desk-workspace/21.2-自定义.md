# 21.2 自定义

工作区自定义分为两大模式：
- **只读模式**：仅查看工作区内容。
- **编辑模式**：允许对工作区进行自定义配置。

进入编辑模式后，用户可通过侧边栏或页面内操作进行配置。

## 一、工作区整体自定义（通过侧边栏操作）

### 1. 编辑工作区详情
- 可修改内容：**标题、图标、父/子级关系、公开/私有设置**。
- 操作路径：侧边栏 → 选择工作区 → 编辑属性。

### 2. 复制工作区
- 功能：复制现有工作区，并可按需增删区块。
- 用途：快速创建类似结构的工作区。

### 3. 删除工作区
- 权限：仅允许删除用户有权限的工作区。

### 4. 隐藏工作区
- 私有工作区：用户可自行隐藏。
- 公开工作区：仅**工作区管理员**可隐藏。

### 5. 调整工作区位置
- 操作：通过侧边栏**拖拽"工作区，调整其顺序或层级关系。
- 注意：侧边栏的修改会"自动保存"，无需手动点击保存。

## 二、工作区页面内容自定义（区块级配置）

### 1. 添加新区块
- 操作：进入编辑模式 → 点击"添加区块" → 选择区块类型并配置内容。

### 2. 编辑现有区块
- 操作：点击区块右上角的"编辑"图标，修改内容或属性。

### 3. 删除区块
- 操作：编辑区块时选择"删除"选项，或直接移除区块。

### 4. 调整区块宽度
- 操作：拖拽区块边缘，调整其占位宽度。

### 5. 重新排列区块
- 操作：拖拽区块标题栏，改变其在页面中的顺序。

## 注意事项
- **保存机制**：页面区块的修改需手动点击"保存"按钮；侧边栏的修改自动生效。
- **权限控制**：部分操作（如隐藏公开工作区）需更高权限。
- **区块类型**：支持多种内置区块（如图表、快捷链接、列表视图等），具体配置参考"区块（Blocks）"文档。
