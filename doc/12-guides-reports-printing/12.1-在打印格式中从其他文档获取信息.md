# 12.1 在打印格式中从其他文档获取信息

## 简介
在打印格式中，您可以从另一个文档获取数据。例如，如果您在销售发票中有一个名为 `sales_order` 的字段，那么您可以使用 `frappe.get_doc` 来获取销售订单的详细信息。

## 代码示例

```jinja2
{% set sales_order_doc = frappe.get_doc("Sales Order", sales_order) %}

{{ sales_order_doc.customer }}
```

## 详细说明

### 功能用途
这段文档指导开发者如何在 ERPNext 或 Frappe 框架的打印格式模板中，通过 Jinja 模板语法访问和显示相关联文档的字段值。

### 关键方法
- 使用 `frappe.get_doc` 方法，该方法接受两个参数：
  - 目标文档的类型（DocType）
  - 文档名称（name）

### 示例场景
假设当前文档是"销售发票"，并且它有一个链接字段指向某个"销售订单"。示例代码演示了如何获取这个关联的"销售订单"文档，并打印出该订单上的"客户"字段值。

### 步骤解析
1. 使用 `{% set sales_order_doc = frappe.get_doc("Sales Order", sales_order) %}` 创建变量
2. 通过 `{{ sales_order_doc.customer }}` 输出相关文档的字段值

这种技术在处理需要显示关联文档信息的打印格式时非常有用。
