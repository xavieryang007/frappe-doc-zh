# 12.2 在哪里找到标准打印格式

## 简介
标准打印格式是根据文档类型（DocType）的布局自动生成的。用户可以通过多种方式对这些标准格式进行自定义和调整。

## 标准打印格式的来源

- 标准打印格式是**根据文档类型（DocType）的布局自动生成**的
- 这些格式不需要单独的模板文件，而是基于表单布局动态生成

## 自定义标准打印格式的方法

### 方法一：通过"自定义表单视图"调整

1. **路径**：进入 *设置 (Setup) > 自定义 (Customize) > 自定义表单视图 (Customize Form View)*

2. **可执行的操作**：
   - **重新排列字段**：通过拖放字段来调整显示顺序
   - **添加静态元素**：通过添加"HTML"类型的字段，并在"选项"中插入HTML代码
   - **隐藏字段**：设置字段的 **Print Hide**（打印时隐藏）属性

### 方法二：基于打印格式创建新布局

由于标准打印格式**没有预生成的模板**，用户需要**从头开始创建新模板**。

1. **路径**：进入 *设置 > 打印与品牌 (Printing and Branding) > 打印格式 (Print Format)*

2. **技术基础**：
   - 使用 **Jinja 模板语言** 进行模板设计
   - 使用 **Bootstrap CSS 框架** 进行布局设计

3. **实用技巧**：
   - 可**导入标准模板宏**，快速构建打印格式
   - **示例代码**（在模板中插入标准页眉）：
     ```jinja
     {%- from "templates/print_formats/standard_macros.html" import add_header -%}
     {{ add_header() }}
     ```

## 重要提示

- 标准打印格式基于文档类型的字段布局自动生成
- 如需更复杂的自定义，建议创建新的打印格式模板
- 可以使用标准模板宏来保持格式的一致性
- 支持通过CSS和HTML实现高度自定义的打印布局

这种机制使得用户既可以快速使用标准格式，也能够根据具体需求进行深度定制。
