# Web表单设置 (Web Form Settings)

## 24.2 设置

### 📋 Web 表单设置

Web 表单设置允许您对表单进行多种自定义，以满足具体业务需求。以下是可配置的主要项目：

### 1. 字段布局（Fields Layout）
- 可在字段表中添加"列分隔符"（Column Break）和"节分隔符"（Section Break）来调整表单布局。

### 2. 多步骤 Web 表单（Multi-Step Webform）
- 可通过插入"分页符"（Page Break）将长表单分为多个页面，提升用户体验。
- 最多支持 9 个分页符，即最多 10 个页面。

### 3. 提交按钮标签（Submit Button Label）
- 可自定义提交按钮的显示文本。

### 4. 横幅图片（Banner Image）
- 支持在 Web 表单顶部添加横幅图片。

### 5. 面包屑导航（Breadcrumbs）
- 可通过 JSON 对象自定义面包屑结构，仅在启用"列表视图"时显示。

### 6. 提交后页面（After Submit Page）
- 可自定义提交成功后的标题和消息。
- 支持设置成功后自动跳转的 URL（`success_url` 字段），5 秒后跳转。

### 7. 自定义 CSS
- 允许添加自定义 CSS 样式，用于美化表单外观。
- 示例代码已提供，包括修改头部、表单区域样式等。

### 8. 客户端脚本（Client Script）
- 支持通过 JavaScript 编写客户端逻辑，常用方法包括：
  - `frappe.web_form.on()`：监听字段变化。
  - `frappe.web_form.get_value()` / `set_value()`：获取或设置字段值。
  - `frappe.web_form.validate`：自定义提交前验证。
  - `frappe.web_form.set_df_property()`：动态修改字段属性（如隐藏、禁用）。
  - `frappe.web_form.after_load`：表单加载完成后执行初始化脚本。

#### 示例脚本：
- **字段值重置**：当金额小于 1000 时提示并重置。
- **自定义验证**：检查金额是否达标。
- **条件隐藏字段**：根据金额隐藏/显示费率字段。
- **启动提示**：表单加载后显示提示信息。

### 💡 使用建议
- 所有设置均可在 Web 表单文档的 **"自定义"** 标签页中找到。
- 多步骤表单适合字段较多的情况，提升填写效率。
- 客户端脚本可实现复杂交互逻辑，但需注意兼容性与性能。
