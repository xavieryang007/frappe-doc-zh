# 27.17 bench generate-pot-file

## **用法**
```bash
bench generate-pot-file [选项]
```

## **描述**
该命令用于生成翻译模板文件（POT 文件），其中包含应用程序中所有可翻译的字符串（但不包含翻译内容）。模板文件将写入到 `[app_module]/locale/main.pot` 文件中。

对于许多应用程序，此命令会通过 GitHub Actions 定期自动运行。

## **选项**
- `--app`：此参数允许您指定一个应用程序，如果只想为某个应用程序生成 POT 文件。默认情况下，会为所有应用程序生成。

## **示例**
**重新生成 ERPNext 的 POT 文件**：
```bash
bench generate-pot-file --app erpnext
```
这将从 `apps/erpnext` 中提取所有可翻译字符串，并将其写入 `apps/erpnext/erpnext/locale/main.pot`。

## **补充说明**
- 该命令主要用于国际化和本地化工作，帮助开发者提取代码中需要翻译的文本。
- 生成的 POT 文件可以用作基础，通过工具（如 Poedit）转换为特定语言的 PO 文件，最终编译为 MO 文件供应用程序使用。
