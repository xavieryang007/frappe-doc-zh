# 27.18 bench migrate-csv-to-po

## **用法**
```bash
bench migrate-csv-to-po [选项]
```

## **前提条件**
确保您的应用已经有一个 POT 文件。如果没有，请先使用以下命令生成：
```bash
bench generate-pot-file
```

## **描述**
此命令从现有的 CSV 翻译文件中读取翻译内容，与现有的 POT 文件进行匹配，并将结果写入 `[app_module]/locale/[locale].po`。

该命令主要用于从旧的翻译系统迁移到新系统时使用。

## **选项**
- `--app`：指定要迁移翻译的应用名称。默认为所有应用。
- `--locale`：指定要迁移的语言区域（如 `de`、`fr`）。默认为所有语言区域。

## **示例**
**迁移 ERPNext 的德语翻译从 CSV 到 PO**：
```bash
bench migrate-csv-to-po --app erpnext --locale de
```

该命令会从 `apps/erpnext/erpnext/translations/de.csv` 读取翻译，从 `apps/erpnext/erpnext/locale/main.pot` 读取可翻译字符串，并将结果保存到 `apps/erpnext/erpnext/locale/de.po`。如果迁移结果符合预期，可以删除原有的 CSV 文件。
