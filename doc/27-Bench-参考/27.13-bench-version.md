# 27.13 bench version

## **用法**
```bash
bench version [选项]
```

## **描述**
`version` 命令显示当前 bench 目录中安装的所有应用程序的编译信息。您可以选择首选的输出格式：纯文本、JSON 或 ASCII 表格。

`--format plain` 选项以纯文本格式显示版本信息，类似于 `bench version`，但包含额外信息。

```bash
$ bench version --format plain
erpnext 14.0.3 version-14 (4e88dcf)
frappe 14.0.1 version-14 (f8ec3d7)
```

`--format json` 选项以格式化的 JSON 字符串显示版本信息。如果您在 bench CLI 之上构建工具，这特别有用。

```bash
$ bench version --format json
[
 {
 "app": "erpnext",
 "branch": "version-14",
 "commit": "4e88dcf",
 "version": "14.0.3"
 },
 {
 "app": "frappe",
 "branch": "version-14",
 "commit": "ef0a5e9",
 "version": "14.0.1"
 }
]
```

`--format table` 选项以 ASCII 表格格式显示版本信息。

```bash
$ bench version --format table
+-------------------------+------------+------------------------------+---------+
| App | Version | Branch | Commit |
+-------------------------+------------+------------------------------+---------+
| erpnext | 14.0.3 | version-14 | 4e88dcf |
| frappe | 14.0.1 | version-14 | f8ec3d7 |
+-------------------------+------------+------------------------------+---------+
```

## **选项**
- `-f, --format`：选择显示当前 bench 中安装应用程序版本的格式。可用选项为 "plain"、"table"、"json"、"legacy"。默认值为 "legacy"。

## **示例**

1. **获取当前 bench 上安装应用程序的人类可读信息，包含提交消息**：
   ```bash
   bench version --format plain
   ```

2. **以 JSON 格式获取 bench 版本信息**：
   ```bash
   bench version -f json
   ```
