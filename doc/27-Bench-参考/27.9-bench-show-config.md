# 27.9 bench show-config

## **用法**
```bash
bench show-config [选项]
```

## **描述**
您的站点应用的配置是 bench 目录的 `common_site_config.json` 和站点自身的 `site_config.json` 组合应用的结果。Bench 提供了一个界面来查看站点的已应用 Frappe 配置值。您可以选择以表格或 JSON 格式查看此信息。

有关站点配置和了解键值优先级的更多信息，请参阅[相关文档](https://docs.frappe.io/framework/user/en/bench/reference/show-config)。

## **标志**
- `-f, --format`：选择列出指定站点上已安装应用的格式，选项为 "text" 和 "json"。默认为 "text"。

## **示例**

1. **以 JSON 格式显示所有站点的站点配置**：
   ```bash
   bench --site all show-config -f json
   ```

2. **以表格形式显示站点配置**：
   ```bash
   bench --site {site} show-config --format text
   ```
   或
   ```bash
   bench --site {site} show-config -f text
   ```
   或
   ```bash
   bench --site {site} show-config
   ```

## **说明**
- 该命令帮助用户查看当前站点的有效配置，这些配置来自 `common_site_config.json`（全局配置）和 `site_config.json`（站点特定配置）的合并结果。
- 使用 `--format` 标志可以灵活地以人类可读的表格形式或机器可读的 JSON 格式输出配置信息。
- 在管理多站点环境或调试配置问题时，此命令非常有用。
