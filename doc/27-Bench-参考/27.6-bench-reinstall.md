# 27.6 bench reinstall

## **用法**
```bash
bench reinstall [选项]
```

## **描述**
重新安装一个站点，使用当前已安装的应用程序。这将清除该站点的所有数据并重新开始。此操作被视为破坏性操作，因此默认包含交互式确认提示。

> **注意**：此功能目前仅适用于 MariaDB 站点。未来可能会扩展支持 PostgreSQL。

## **选项**
- `--admin-password`：重新安装后站点的管理员密码。
- `--mariadb-root-username`：MariaDB 的 root 用户名。
- `--mariadb-root-password`：MariaDB 的 root 密码。

## **标志**
- `--yes`：跳过重新安装的确认提示。

## **示例**

1. **重新安装一个站点，跳过以下提示**：
   - 操作确认
   - MariaDB root 密码
   - 管理员密码

   ```bash
   bench reinstall {站点名} --yes --mariadb-root-password {数据库密码} --admin-password {管理员密码}
   ```

2. **使用具有 DBMS SUPER 权限的替代用户重新安装站点**：
   ```bash
   bench reinstall --mariadb-root-username {数据库用户} --mariadb-root-password {数据库密码}
   ```
