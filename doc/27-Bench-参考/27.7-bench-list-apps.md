# 27.7 bench list-apps

## **用法**
```bash
bench list-apps [选项]
```

## **描述**
列出指定站点上安装的所有 Frappe 应用程序。此命令显示的信息来自"已安装应用程序"（Installed Applications）DocType，该类型跟踪站点迁移到的最新应用版本。全局默认的 `installed_apps` 用作备用。

多站点支持已在版本 13 中添加。要查看所有站点的摘要，请使用站点值为 `all` 运行命令。

## **选项**
- `--format`, `-f`：选择列出指定站点上已安装应用的格式，选项为 "text" 和 "json"。默认为 "json"。

## **示例**

1. **列出所有站点上安装的应用**：
   ```bash
   bench --site all list-apps
   ```

2. **以可解析的 JSON 格式列出所有站点上安装的应用**：
   ```bash
   bench --site all list-apps --format json
   ```

3. **以文本格式列出特定站点上安装的应用**：
   ```bash
   bench --site {site} list-apps --format text
   
   # 或者使用简写
   bench --site {site} list-apps -f text
   
   # 或者使用默认格式（JSON）
   bench --site {site} list-apps
   ```

## **说明**
- 该命令主要用于查看在 Frappe 框架环境中已安装的应用程序列表。
- 使用 `--site all` 可以查看所有站点的应用安装情况。
- 输出格式可以选择 JSON（默认，便于机器解析）或文本（便于人工阅读）。
