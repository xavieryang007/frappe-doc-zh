# 17.2 Postgres数据库设置

## 设置命令

要使用Postgres作为新站点的数据库，可以使用以下命令：

```bash
bench new-site <站点名称> --db-type postgres
```

## 注意事项

- 确保系统中已安装**Postgres 9或更高版本**。

## 关键点总结

- Postgres数据库设置仅需在创建新站点时通过`--db-type`参数指定。
- 无需额外配置或修改现有设置，Frappe框架会自动处理与Postgres的兼容性。

## 补充说明

- 网页内容中未提及Postgres的具体配置细节（如连接参数、权限设置等），仅提供了基本的创建站点命令。
- 如果需要进一步优化或调整Postgres配置，建议参考Postgres官方文档或Frappe框架的部署指南。
