# 单一文档类型

"单一文档类型"是一种在数据库中仅有一个实例的文档类型。它适用于存储那些不需要多条记录的系统设置等内容。

## 示例代码

```python
>>> settings = frappe.get_doc('System Settings')
>>> settings.notification_frequency
'Daily'
```

## 数据库存储结构

单一文档类型的数据存储在 `tabSingles` 表中，每个属性都有其对应的记录。

### 表结构包含以下列：

- **doctype**：文档类型
- **field**：字段名
- **value**：字段值
