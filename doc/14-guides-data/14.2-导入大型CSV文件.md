# 14.2 导入大型CSV文件 (Import Large CSV File)

## 导入大型CSV文件指南

### 1. **使用场景**
- 当需要导入非常大的CSV文件时，使用Web界面可能会因超时而失败。
- 推荐使用`bench`命令行工具中的`import-csv`命令，以避免超时问题。

### 2. **命令语法**
```bash
bench --site <站点名称> import-csv <CSV文件路径>
```
**示例**：
```bash
bench --site test.erpnext.com import-csv ~/Downloads/Activity_Type.csv
```

### 3. **命令选项**
`import-csv`命令支持以下选项：
- `--only-insert`：仅插入新记录，不覆盖现有记录。
- `--submit-after-import`：导入后自动提交文档。
- `--ignore-encoding-errors`：在转换为Unicode时忽略编码错误。
- `--help`：显示帮助信息。

### 4. **帮助信息**
可以通过以下命令查看完整的帮助信息：
```bash
bench import-csv --help
```

### 5. **注意事项**
- 该命令基于Frappe框架的**数据导入工具**，适用于大规模数据导入。
- 确保CSV文件的格式与目标DocType的字段匹配，否则导入可能失败。
- 如果导入过程中遇到编码问题，可以使用`--ignore-encoding-errors`选项跳过错误。

## 总结
使用`bench import-csv`命令是处理大型CSV文件导入的最佳实践，能够有效避免Web界面的超时限制，并提供灵活的导入选项。如果您需要进一步调整导入行为（如仅插入或不覆盖记录），可以通过命令选项进行配置。
