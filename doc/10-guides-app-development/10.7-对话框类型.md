# 10.7 对话框类型

Frappe框架提供了一组标准对话框，在开发过程中非常实用。以下是各类对话框的详细说明：

## 1. 警告对话框（Alert Dialog）

**用途**：用于显示非阻塞性消息。

**参数**：
- `txt`：要在警告对话框中显示的消息
- `seconds`：消息显示的持续时间（默认3秒）

**示例代码**：
```javascript
show_alert('Hi, do you have a new message', 5);
```

---

## 2. 提示对话框（Prompt Dialog）

**用途**：用于从用户收集数据。

**参数**：
- `fields`：包含字段对象的列表
- `callback`：处理对话框数据的函数
- `title`：对话框标题
- `primary_label`：主要按钮的标签

**示例代码**：
```javascript
frappe.prompt([
    {'fieldname': 'birth', 'fieldtype': 'Date', 'label': 'Birth Date', 'reqd': 1}
], function(values){
    show_alert(values, 5);
}, 'Age verification', 'Subscribe me');
```

---

## 3. 确认对话框（Confirm Dialog）

**用途**：在执行操作前获取用户确认。

**参数**：
- `message`：对话框中显示的消息
- `onyes`：确认时的回调函数
- `oncancel`：取消时的回调函数

**示例代码**：
```javascript
frappe.confirm(
    'Are you sure to leave this page?',
    function(){ window.close(); },
    function(){ show_alert('Thanks for continue here!'); }
);
```

---

## 4. 消息打印（Message Print）

**用途**：向用户显示信息（支持HTML格式）。

**参数**：
- `message`：要显示的消息（可以是HTML字符串）
- `title`：对话框标题

**示例代码**：
```javascript
msgprint("Server Status" + "<br><br>" + 
    "<ul>" +
    "<li>28% Memory</li>" +
    "<li>12% Processor</li>" +
    "<li>0.3% Disk</li>" +
    "</ul>", 'Server Info');
```

---

## 5. 自定义对话框（Custom Dialog）

**用途**：通过扩展`frappe.ui.Dialog`创建自定义对话框。

**示例代码**：
```javascript
var d = new frappe.ui.Dialog({
    'fields': [
        {'fieldname': 'ht', 'fieldtype': 'HTML'},
        {'fieldname': 'today', 'fieldtype': 'Date', 'default': frappe.datetime.nowdate()}
    ],
    primary_action: function(){
        d.hide();
        show_alert(d.get_values());
    }
});

d.fields_dict.ht.$wrapper.html('Hello World');
d.show();
```

## 总结

Frappe框架提供了五种主要对话框类型：
- **警告对话框**：简单消息提示
- **提示对话框**：数据收集
- **确认对话框**：用户确认
- **消息打印**：信息展示（支持HTML）
- **自定义对话框**：完全自定义的对话框

每种对话框都有特定的使用场景和参数配置，您可以根据实际需求选择合适的类型。
